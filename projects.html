Programming Style Problems

Link to Source

During my 3rd year at UCI I took a class where we went over many different styles and features of different programming languages and patterns. To practice with languages and styles, I reimplemented most of the Python 3 examples in the pictured textbook in C++ and Java, usually about 2 examples each week. Each week has its own folder in the repl directory.

I listed this here for 2 reasons:

  • 1) I challenged myself to take a unique (potentially impractical) solution each time I worked on a new example.

  • 2) In each example, I had to use a lot of neat tricks to get around C++ static typing (including the first time I ever used constexpr). I think it is a good sign of my ingenuity as programmer and my ability to solve unusual problems.
